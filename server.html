How to Set Up a Minecraft Server

Welcome to our guide on creating your own Minecraft server! Follow these steps:

Step 1: Create server using aternos

Visit the Aternos website and create an account!

Create a server either a bedrock or java (we'll talk about creating a crossplay server after). You can name it anything you want but in this example I will call it "Toddserver"

Go to the software tab then click on either paper/bukkit or spigot... after that click install, then go to the plugins tab and search:

  1. Basic Quests
  2. Better Structures
  3. Chest Shop
  4. ClickTPA
  5. EssentialsX
  6. Automatic-Broadcast
  7. Geyser
  8. and GeyserSkinManager
  9. GSIT
  10. Luckperms
  11. Multiverse Core
  12. Mythic Mobs
  13. PlaceholderAPI
  14. PowerRanks
  15. SimpleScore
  16. Vault
  17. World Edit
  18. World Gaurd
  19. Znpcs

Step 2: Edit the Plugins

Click the Server tab and then start the server.

Check if your plugins are installed by typing /plugins in the console.

Now turn off the server and go to the files tab and click plugins! Now scroll through and edit the plugins. on some of the plugins like EssentialsX I recommend Kassi sora for them. And for geyser if you chose to use it for crossplay use this video by badger10go also for power ranks use this video by KilanFluffySheep (also for power ranks you dont have to go in game to do it just do it in console) And after you finish all of that go to the next step.

Step 3: Start the server

In the server tab, click the start button to start the server of course. Now wait until it starts and then join...
If you are on Java use the ip but if you are on Bedrock use the port and ip

Step 4: Configure Server Properties

Open the server.properties file in the files tab and then edit it all to your liking.

Such as server name, difficulty, and more.

Step 5: Geyser

If you want bedrock or mcpe players to join your server, you'll need to set up geyser. Refer to the video listed above for instructions.

Step 6: Play it!

That's it! You now have your own Minecraft server up and running. Enjoy!








Creating a Minecraft server using Minecraft server software

Welcome to our guide on creating your own Minecraft server! Follow these steps:

Step 1: Download the Minecraft Server Software

Visit the official Minecraft website and download the minecraft_server.1.XX.X.jar file (where the X's represent the current version number).

Create a folder on your desktop to keep all your server files. You can name it "Minecraft Server."

Drag the downloaded .jar file into the newly created folder.

Step 2: Run the Server

Open the command prompt (you can search for it in the Start menu).

Check if Java is installed by typing java -version.

Change the directory to where you have the Minecraft Server folder and the .jar file. You can do this by typing:

cd C:\Users\Admin\Desktop\Minecraft Server

Replace the path with the actual location of your folder.

Step 3: Start the Server

In the command prompt, type the following (replace "name of your jar" with the actual name of your .jar file):

java -jar "name of your jar".jar

You might see a "failed to load properties" message; don't worry, this is normal.

Step 4: Configure Server Properties

Open the server.properties file in a text editor (located in your Minecraft Server folder).

Customize settings such as server name, difficulty, and more.

Step 5: Port Forwarding

If you want others to join your server, you'll need to set up port forwarding on your router. Refer to your router's documentation for instructions.

That's it! You now have your own Minecraft server up and running. Enjoy!
